Build a complete Flutter mobile app prototype called WHAMI.

App meaning:
WHAMI = “Where Am I?”
Tagline: “Verified by the real world.”

Core concept:
This is not a normal navigation app. It is a Verified Navigation / Position Trust Engine. The app compares multiple independent position opinions and tells the user whether GPS is believable. GPS is only one witness. WHAMI should visually compare GPS with landmark/seamap, magnetic field, IMU movement, and sextant/sky validation.

Important:
This is a working UI prototype with proper mock data only. Do not use real GPS, real ARCore, real maps, or real sensors yet. Simulate all data through a clean mock data layer. The app should run without API keys, backend, or internet dependency.

Tech stack:

* Flutter
* Clean folder structure
* Responsive mobile UI
* Use mock repositories/services
* No backend
* No Firebase
* No paid map API
* Use CustomPaint or a local mock map canvas for the map screen
* Keep code easy to replace with real APIs later

Design style:

* Premium, serious, navigation/security style
* Dark header with WHAMI logo text
* White/light background cards
* Navy blue, amber/yellow, black, red, green, purple marker colors
* Rounded cards
* Clear trust score badge
* Modern app UI
* Must look good on Android phones

Main colors:

* GPS: Blue
